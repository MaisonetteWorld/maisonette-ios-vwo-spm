## Installation
VWO is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```
pod 'VWO', '~> 2.0.0-RC2'
```

## iOS Version Support

This library supports iOS version 8.0 and above.
